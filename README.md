## 如何使用

1. 先从对应岗位高频主题开始，例如 Unity 客户端优先看 C++、Unity / C#、数据结构与算法、游戏客户端工程。
2. 每个问题先尝试自己口述 1 分钟，再展开答案查漏补缺。
3. 对“区别是什么”“为什么”“如何设计/优化/排查”这类题，重点记回答结构：结论、原理、场景、坑点。
4. 对算法和系统设计题，额外关注复杂度、边界条件、生命周期、线程安全和资源释放。
